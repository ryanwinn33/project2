Endpoints

  • GET all disasters by year
    • "/api/disasters"
  • GET all disasters with monthly data
    • "/api/disastermonth"
  • GET all disasters within a given year (range 2016-2019)
    • "/api/disasters/(enter year)"
    • Example: "/api/disasters/2016"
  • GET all disasters from a specific state
    • "/api/disasters/(enter state name)"
    • Example: "/api/disasters/Oregon"
